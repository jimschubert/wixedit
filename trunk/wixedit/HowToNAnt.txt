How to build WixEdit with NAnt:

Download NAnt 0.84

Make sure NAnt.exe is added to your PATH variable.

Open a command prompt and make sure all visual studio paths are added to your PATH variable.
(Run C:\Program Files\Microsoft Visual Studio .NET 2003\Vc7\bin\vsvars32.bat in the command prompt)

Browse to the directory where the WixEdit.build is, and start nant.
Like:
C:\WixEdit>NAnt.exe

The binaries can be found in the build sub directory.
